# REDIMAG - Sitio institucional con catálogo consultivo

Sitio web estático para REDIMAG, marca de insumos veterinarios para animales domésticos y de producción.

## Estructura

```text
/
├── index.html
├── catalogo.html
├── producto.html
├── nosotros.html
├── contacto.html
├── css/
│   └── styles.css
├── js/
│   └── main.js
├── assets/
│   ├── logo.png
│   ├── CowGrass wide.png
│   ├── CowGrass.png
│   ├── cow.png
│   └── img/
└── README.md
```

## Uso local

Abre `index.html` directamente en el navegador. También puedes navegar a `catalogo.html`, `nosotros.html` y `contacto.html`. No requiere servidor, backend, frameworks ni dependencias de instalación.

## Funcionalidades

- Home institucional reducido con hero, resumen de marca y accesos por línea.
- Vista de catálogo completa con filtros tipo e-commerce.
- Vista interna de producto con CTA de WhatsApp por producto.
- Vista Nosotros con landing institucional.
- Vista Contacto con formulario visual.
- Submenú de líneas de producto desde la navegación principal.
- Topbar delgado con WhatsApp cliqueable.
- Header sticky con menú hamburguesa en mobile.
- Enlaces dinámicos a WhatsApp con mensajes prellenados.
- Formulario visual con simulación de envío.
- Diseño responsive para mobile, tablet y desktop.

## Nota

El sitio no incluye carrito, checkout, pasarela de pagos ni lógica de compra. La conversión principal es la consulta por WhatsApp.
